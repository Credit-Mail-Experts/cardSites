# CME Card Sites

This site acts as a template for all CME card websites.

It allows programmers in the office to easily make changes across 5 separate sites.

### Commit Message Titles
Must be one of the following:
* **feat**: A new feature
* **fix**: A bug fix
* **style**: css styling changes
* **refactor**: A code change that neither fixes a bug or adds a feature
* **perf**: A code change that improves performance
* **docs**: Documentation only changes things like comments
* **minor**: A few small change that has little effect mostly needed when keep the repo head cleared
